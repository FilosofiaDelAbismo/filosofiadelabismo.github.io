Guía interna · sustituir tus enlaces de afiliado
Cuando te registres en cada programa, sigue estos pasos para actualizar los links en el código.
Paso 1
Regístrate en los programas de afiliados
Cada herramienta tiene su propio programa. Busca "programa de afiliados" o "affiliate program" en su web. La aprobación suele ser inmediata para sitios de herramientas IA. Te darán un link único del tipo https://jasper.ai?fpr=TU_ID.
Paso 2
Abre partners.html y localiza la card
Cada tarjeta de afiliado tiene un <a href="..."> con la URL actual. Reemplaza solo el contenido del href con tu link.
href="https://jasper.ai?fpr=TU_ID_AQUI"
Paso 3
Programas habituales y enlaces de registro
Referencia rápida:
Jasper AI · jasper.ai/affiliate
ElevenLabs · elevenlabs.io/affiliate
Synthesia · synthesia.io/partners
Pictory · pictory.ai/affiliate
Canva · canva.com/affiliates
TubeBuddy · tubebuddy.com/affiliate
Semrush · semrush.com/lp/affiliate
MailerLite · mailerlite.com/referral
Hetzner · hetzner.com/referral (20€ crédito)
Paso 4
Sube el archivo a GitHub y listo
Haz commit y push del archivo actualizado. Los cambios se publican en menos de 2 minutos en GitHub Pages.
git add partners.html && git commit -m "update affiliate links" && git push
