Guía para sustituir tus enlaces de afiliado
Cuando te registres en cada programa, sigue estos pasos para actualizar los links en el código. Un Ctrl+F por cada línea numerada.
Paso 1
Regístrate en los programas de afiliados
Cada herramienta tiene su propio programa. Busca "programa de afiliados" o "affiliate program" en su web. La aprobación suele ser inmediata para sitios de herramientas IA. Te darán un link único del tipo https://jasper.ai?fpr=TU_ID
Paso 2
Abre partners.html en tu editor y busca las LÍNEAS numeradas
Cada tarjeta de afiliado tiene un comentario HTML indicando qué sustituir. Busca con Ctrl+F el comentario exacto:
Paso 3
Sustituye el href del enlace en la línea siguiente
Justo después del comentario, hay un tag <a href="...">. Reemplaza solo el contenido del href con tu link de afiliado:
href="https://jasper.ai?fpr=TU_ID_AQUÍ"
Paso 4
Índice de las 12 líneas a sustituir
Referencia rápida de todas las líneas numeradas en el código:
LÍNEA 1 → Claude Pro (sin afiliado, URL directa)
LÍNEA 2 → Jasper AI — jasper.ai/affiliate
LÍNEA 3 → ElevenLabs — elevenlabs.io/affiliate
LÍNEA 4 → Synthesia — synthesia.io/partners
LÍNEA 5 → Pictory — pictory.ai/affiliate
LÍNEA 6 → Canva — canva.com/affiliates
LÍNEA 7 → TubeBuddy — tubebuddy.com/affiliate
LÍNEA 8 → Semrush — semrush.com/lp/affiliate
LÍNEA 9 → MailerLite — mailerlite.com/referral
LÍNEA 10 → Hetzner — hetzner.com/referral (20€ crédito)
LÍNEA 11 → Cloudflare (sin programa oficial)
LÍNEA 12 → Gumroad (sin programa oficial)
Paso 5
Sube el archivo a GitHub y listo
Haz commit y push del archivo partners.html actualizado. Los cambios se publican en menos de 2 minutos en GitHub Pages.
git add partners.html && git commit -m "update affiliate links" && git push
